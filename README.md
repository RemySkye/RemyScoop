# RemyScoop

Welcome to my personal Scoop bucket! This repository contains custom application manifests that I maintain for my own use.

## Usage

To add this bucket to your local Scoop installation, run:

```powershell
scoop bucket add RemyScoop https://github.com/RemySkye/RemyScoop.git
```

Then you can install the packages here normally:

```powershell
scoop install RemyScoop/kushview-element
```

## Available Packages

Here are the custom packages currently maintained in this bucket:

* **kushview-element** - A flexible plugin host and routing matrix by Kushview.
* **hesuvi** - Headphone Surround Virtualizations for Equalizer APO.
* **office-tool-plus-with-runtime** - Office Tool Plus is a tool for managing, downloading and installing Microsoft Office.
* **real-video-enhancer** - An open-source video resolution enhancer utilizing AI upscaling.
* **safe-mode-launcher** - Safe Mode Launcher is a freeware GUI app to swiftly restart your computer in Windows Safe Mode.
* **tcpoptimizer** - SG TCP Optimizer. A free, easy Windows program that provides an intuitive interface for tuning and optimizing your Internet connection.
* **vb-cable-pack43** - VB-Audio Virtual Cable. A virtual audio device working as a virtual audio cable (Standard Version / Pack 43).
* **vb-cable-pack45** - VB-Audio Virtual Cable (Pack 45). The newer version that inherently includes 16-channel endpoint support.
